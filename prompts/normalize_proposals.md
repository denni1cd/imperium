# Normalize Strategy Proposals

Compare the independent strategy proposals and create a proposal-phase `ClaimRegister`.

Normalize only decision-relevant proposed actions, assumptions, forecasts, risks, and tradeoffs. Preserve each source proposal and member identifier. Separate factual, interpretive, and value disagreements. Assign materiality from the expected effect on the final recommendation, not rhetorical emphasis. Return exactly one proposal-phase `ClaimRegister`.
